# Syntecxhub Data Cleaning Utility

This project was completed as part of my Data Science Internship at Syntecxhub.

## Objective
To clean and preprocess raw data using Python and Pandas.

## Features
- Handling missing values
- Removing duplicate records
- Standardizing column names
- Converting date columns
- Exporting cleaned dataset

## Tools Used
- Python
- Pandas
- Google Colab

## Output
The cleaned dataset is saved as `cleaned_data.csv`.
